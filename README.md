 ### Careful, now. If you have epilepsy or you don't like super fast changing colors and pixels, i'd rather you not run this...

## Run on Linux

Run these commands in a terminal:
 
```bash

curl -L https://github.com/NullExceptionDialog/Linux-TurkishFlagWave/raw/main/flagwave.zip -o flagwave.zip
unzip flagwave.zip -d flagwave/
chmod +x flagwave/flagwave.sh

```
After that, run this:
``` bash
cd flagwave
./flagwave.sh
```
